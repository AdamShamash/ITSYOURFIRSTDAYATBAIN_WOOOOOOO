# First Day Hype Site

Upload both files to the root of your GitHub repo:

1. index.html
2. freesound_community-dj-airhorn-sound-39405.mp3

Then keep GitHub Pages set to:
- Branch: main
- Folder: / (root)

Important:
Most browsers block sound on first page load until the viewer clicks once. This site tries to play the horn automatically. If the browser blocks it, the site shows a big "Unlock Hype Mode" button. After that click, the horn will play twice on tab switches.
